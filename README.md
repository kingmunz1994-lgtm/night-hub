# Night Hub

> **The central launchpad for the Night ecosystem on Midnight Network.**

Night Hub links every Night app into a single, privacy-first financial ecosystem powered by ZK proofs on [Midnight Network](https://midnight.network).

## Night Score

Every action across the ecosystem earns **Night Score** — a cross-app reputation layer stored in your browser and backed by ZK proofs on-chain.

## Built With

- **Midnight Network** — ZK blockchain with Compact smart contracts
- **ZK Proofs** — privacy-preserving verification without revealing sensitive data
- **DUST / NIGHT tokens** — native ecosystem currencies
