The Technology

How Midnight makes this possible

🔐
Compact Circuits
Every app's logic lives in a Compact v0.20 contract on Midnight. Circuits generate ZK proofs client-side — the chain sees only a proof, never your data.
🪙
NIGHT + DUST
NIGHT is the public utility token. Holding NIGHT generates DUST, which pays for private transaction fees. Stake ADA to a Midnight-enabled pool to earn NIGHT automatically.
⚡
State Channels
Betting rounds, task updates, and marketplace negotiations run off-chain at web speed. Only the final signed root settles on Midnight — no gas per click.
🏗️
Built on Cardano
Midnight is a partner chain to Cardano. Cardano SPOs run Midnight validators — ADA delegators to those pools earn both ADA staking rewards and NIGHT.
